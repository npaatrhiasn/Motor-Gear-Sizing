# Electric Forklift Motor-Reducer Sizing Project

This repository contains the complete design and sizing documentation for a motor-reducer unit intended for an electric forklift. The work covers energy needs assessment, motor and battery selection, gear reduction sizing, and CAD modeling of the housing and shafts.

## Project Overview

The goal is to dimension a motor-reducer system for an electric forklift capable of performing a typical daily work cycle. The project is divided into four deliverables (Livrables), each addressing a specific aspect of the design.

## Repository Contents

| File | Description |
|------|-------------|
| `Livrable 1.pdf` | Energy requirement analysis: daily energy consumption, average power, maximum torque, and equivalent thermal torque for both driving and lifting phases. |
| `Livrable 2.pdf` | Motor characteristics, transmission ratio, acceleration capacity on flat ground, maximum climbable slope at start, battery technology, voltage, capacity, and estimated mass. |
| `Livrable 3.pdf` | Sizing of gear teeth and the reduction gear unit. |
| `Livrable 5.pdf` | CAD models: functional surfaces of the housing, complete motor-reducer assembly integrated into the provided CAD model, and detailed drawings of the shafts. |

> **Note:** Livrable 4 is not included in this repository.

## Key Results (Summary)

- **Energy needs** determined for a full working day based on a realistic duty cycle.
- **Motor and transmission** selected to meet torque, speed, and thermal constraints.
- **Battery** type, voltage, capacity, and mass estimated.
- **Gear reduction** sized to withstand mechanical loads.
- **3D CAD models** of the housing, assembly, and shaft drawings provided.

## How to Use This Repository

Simply download or view the PDF files for detailed calculations, assumptions, and technical drawings.
